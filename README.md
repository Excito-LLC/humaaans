# Humaaans: Create Human Illustrations With Ease.

## Demo
# ![Demo](https://humaaans.s3.amazonaws.com//demo.gif)

## Get Started
Install from NPM:
```
npm install --save-dev humaaans
```
Import it:
```
import Human from 'humaaans';
```
Use it:
```
<Human head="Afro" torso="Hoodie" bottom="BaggyPants" />
```
